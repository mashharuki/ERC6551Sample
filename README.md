# ERC6551Sample
ERC6551Sample Repo

### ERC6551の可能性

NFTをウォレットの様に振る舞わせることができる可能性あり。

### How to run 

- install

```bash
cd erc6551-sample && yarn
```

- compile

```bash
cd erc6551-sample && yarn compile
```

- test

```bash
cd erc6551-sample && yarn test
```
